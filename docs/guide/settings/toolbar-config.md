# 工具配置

鍵盤**上方 10 格工具列**要放哪些按鈕。設定會寫進皮膚，套用到所有鍵盤類型。

<img src="../screenshots/11-toolbar-config.png" alt="工具配置：編輯 10 格工具列" class="wide-img">

## 怎麼操作

1. 點選上方某一格（高亮為「選中工具」）
2. 點下方工具卡片，依序填入選中的格
3. **清空全部** 或 **還原預設** 可快速重置

> 若佈局為「**九宮格數字 + 面板符號**」，選擇器中**不會顯示**「獨立九宮格數字」「獨立符號面板」（29／30）。

## 預設 10 格

| 槽位 | 工具 |
|:----:|------|
| 1 | 設定面板 |
| 2 | （空白） |
| 3 | （空白） |
| 4 | 中英切換 |
| 5 | 簡繁切換 |
| 6 | 常用語面版 |
| 7 | 剪貼簿面版 |
| 8 | 符號鍵盤 |
| 9 | Emoji 鍵盤 |
| 10 | 收折按鈕 |

## 可選工具一覽

圖示為設計器實際 SVG（淺色主題）。文字型工具（英、简、ㄅ、拼）以鍵帽樣式顯示。

部分簡介請見 [使用指南·工具列](https://ryanwuson.github.io/rime-liur-ios/#/daily-use/toolbar)。
