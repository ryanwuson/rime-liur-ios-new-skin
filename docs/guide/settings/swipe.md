# 滑動與長按

控制鍵盤的上滑、下滑、長按功能，**套用到全部鍵盤**。

- 上方有**總開關**，可先全部開或關
- 下方**分行滑動與長按開關**可針對每一行（Q 列、A 列等）個別設定
