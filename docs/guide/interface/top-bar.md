# 上方工具列

設計器頁面最上方（非鍵盤上的 10 格工具列）。

<img src="../screenshots/02-top-toolbar.png" alt="上方工具列" class="wide-img">

| 圖示 | 按鈕 | 用途 |
|------|------|------|
| <img src="../images/ui/skin-info.svg" class="ui-icon" alt=""> | <a id="編輯皮膚資訊"></a>**編輯皮膚資訊** | 設定皮膚名稱、作者（會寫進導出的檔案） |
| <img src="../images/ui/import-cskin.svg" class="ui-icon" alt=""> | **匯入配置** | 讀取 `.cskin` 繼續編輯（自己的或別人的皮膚皆可） |
| <img src="../images/ui/export-cskin.svg" class="ui-icon" alt=""> | **導出配置** | 下載 `.cskin` 皮膚檔 |
| <img src="../images/ui/book.svg" class="ui-icon" alt=""> | **使用說明** | 查看如何在元書輸入法 App 裡讓皮膚生效 |

匯入／導出詳見 [匯入與導出](import-export.md)。
